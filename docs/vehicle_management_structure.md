# 車両管理 (2).xlsx 構造概要

## 概要
- ファイル: `車両管理 (2).xlsx`
- シート数: 7
- 主用途: 車両のリース/保険/管理情報の一覧と、更新(再リース/新車入替)の進捗管理、終了車両履歴、ETC/レンタカーカード管理

## 共通メモ
- 登録番号は複数列に分割されている構造（例: 地名 / 分類番号(3桁) / かな / 番号(4桁)）。
- 先頭の空列や `Unnamed` 列は、見出しの結合セルや余白列に由来するため実質的に未使用。
- 行数は概数（空行を含む場合あり）。

## シート別

### 1) 車両一覧
- 役割: 主要な車両台帳（稼働中/管理対象）
- 見出し: 1行目
- データ件数: 約52件
- 主要項目:
  - 車両識別: 会社/区分（例: ｸﾗﾊｼ等）、登録番号（分割列）
  - 車両情報: 車種、車台番号
  - 契約情報: 契約開始日/満了日、契約期間、車検満了日、リース料（税抜）、契約番号、リース会社、リース連絡先、明細番号
  - 保険: 保険代理店、保険会社、保険料、自賠責保険、自賠責保険会社
  - 管理: 管理部門、管理担当者、JAF会員証/有効期限
  - メンテ/タイヤ: メンテナンス工場/連絡先、タイヤ交換先/連絡先
  - カード/備品: 給油カード、使用状況、入場許可証、スペアキー
